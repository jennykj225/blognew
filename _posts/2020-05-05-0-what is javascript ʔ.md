# what is javascript ?
## 简介
- 轻量级的脚本语言
- 嵌入式语言
- 宿主环境：浏览器，nodeJS(服务器)
- 面向对象，但也支持其他编程范式（如函数式编程）
- 核心语法：语法构造&标准库
- 浏览器提供的API(浏览器控制，DOM类，web类)
![](https://raw.githubusercontent.com/Sethites/Image/master/JavaScript.png)
## 学习顺序
- 基本语法
- 标准库
- 浏览器 API
- DOM
